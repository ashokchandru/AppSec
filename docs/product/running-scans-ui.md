# Running Scans from the UI

Use the AppSec+ interface to trigger security scans.

## Supported Scan Types
- **SAST:** Static Code Analysis  
- **SCA:** Dependency Vulnerability Scanning  
- **DAST:** Dynamic Application Testing  
- **Container:** Image Security Review  

## Running a Scan
1. Navigate to **Applications**
2. Open an application
3. Go to the **Scans** tab
4. Click **Run Scan**
5. Choose scan type
6. Start scan

## Viewing Scan Progress
The scan detail page shows:
- Status
- Duration
- Issues detected
- Logs (if applicable)
