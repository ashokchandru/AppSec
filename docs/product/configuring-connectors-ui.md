# Configuring Connectors in AppSec+

Connectors integrate AppSec+ with external systems.

## Supported Connectors
- GitHub / GitLab
- Jenkins
- Azure DevOps
- Slack
- Jira

## Adding a Connector
1. Go to **Connectors**
2. Click **Add Connector**
3. Choose integration type
4. Provide required tokens/authentication
5. Test and save

## Connector Status
Each connector shows:
- Active / inactive state
- Last sync time
- Failure messages
