# Connectors Overview

Connectors allow AppSec+ to integrate deeply with your existing ecosystem.

## Categories

### 1. Source Code Management
- GitHub
- GitLab
- Bitbucket

### 2. CI/CD Platforms
- GitHub Actions  
- Jenkins  
- GitLab CI  
- Azure Pipelines  

### 3. Ticketing Systems
- Jira  
- ServiceNow  

### 4. Notification Systems
- Slack  
- Microsoft Teams  

## What Connectors Provide
- Automated scan triggers  
- Import of code metadata  
- Automatic ticket creation  
- Real-time notifications  
- Audit log synchronization  

Connectors help make AppSec+ a part of your secure SDLC.
