# Quickstart Guide

Welcome to the AppSec+ Quickstart guide. This tutorial helps you get up and running with the platform in minutes.

## 1. Create Your Account
Start by signing up and logging into your AppSec+ dashboard.

## 2. Add Your First Application
1. Navigate to **Applications → Add Application**  
2. Provide basic details such as:
   - Application name  
   - Description  
   - Owner / team  

## 3. Configure a Connector (Optional)
Connect CI/CD, SCM, or cloud services to automate scan ingestion.

Supported connectors:
- GitHub
- GitLab
- Azure DevOps
- Jira
- Slack

## 4. Run a Scan
You can trigger scans using:
- UI  
- API  
- CI/CD pipeline  
- Scheduled scans  

## 5. Review Results
Open **Vulnerabilities → Overview** to explore:
- Findings  
- Severity  
- Impacted components  
- Suggested remediation  

## 6. Next Steps
- Set up policies  
- Configure notifications  
- Automate triage
